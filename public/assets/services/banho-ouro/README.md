# Imagens para Banho de Ouro Profissional

## Imagens necessárias:

### 1. Hero Section
- **hero-mobile.webp** - Imagem para mobile (formato: 750x1000px aproximadamente)
  - Sugestão: Joia sendo trabalhada ou peça com banho de ouro reluzente
  
- **hero-desktop.webp** - Imagem para desktop (formato: 800x1000px aproximadamente)
  - Sugestão: Cícero trabalhando no banho de ouro ou equipamento de galvanoplastia

### 2. Galeria Antes/Depois ⭐ IMPORTANTE
Cada caso precisa de 2 imagens (ANTES e DEPOIS):

**Caso 1 - Anel de Prata Transformado:**
- `antes-depois-01-before.webp` - Anel de prata antes do banho
- `antes-depois-01-after.webp` - Anel dourado após o banho

**Caso 2 - Corrente Revitalizada:**
- `antes-depois-02-before.webp` - Corrente com banho desgastado
- `antes-depois-02-after.webp` - Corrente com novo banho

**Caso 3 - Brincos Renovados:**
- `antes-depois-03-before.webp` - Brincos com banho gasto
- `antes-depois-03-after.webp` - Brincos renovados

**Caso 4 - Pulseira Folheada:**
- `antes-depois-04-before.webp` - Pulseira oxidada
- `antes-depois-04-after.webp` - Pulseira reluzente

**Caso 5 - Pingente Antigo:**
- `antes-depois-05-before.webp` - Pingente antigo
- `antes-depois-05-after.webp` - Pingente com brilho renovado

**Caso 6 - Conjunto Completo:**
- `antes-depois-06-before.webp` - Conjunto antes
- `antes-depois-06-after.webp` - Conjunto após banho

**Total: 12 imagens para a galeria (6 casos x 2 imagens cada)**

## Formato recomendado:
- Tipo: WebP (melhor performance)
- Tamanho: 800x800px (quadrado para melhor visualização)
- Qualidade: 85-90%
- Otimização: Comprimir sem perder qualidade visível
- Enquadramento: Centralizar a peça na foto
- Iluminação: Garantir boa iluminação para mostrar o brilho

## Dicas para as fotos:
- Use fundo neutro (branco ou cinza)
- Mesma peça nas duas fotos (antes/depois)
- Mesmo ângulo para facilitar comparação
- Boa iluminação para destacar o brilho do ouro
- Foco nítido na peça
